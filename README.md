# FaceReader Landmarks 可视化工具

虽然FaceReader有各个Landmarks的编号图，但是并没有编号索引。有的时候，想要在四五百个点里面找到某个特定点，是一个不小的挑战。  
索性做了一个基于Web的面部特征点可视化工具，支持2D和3D FaceReader Landmarks数据的交互式展示，可以通过输入Landmarks编号，或者选择特定编号，快速找到对应Landmarks的位置（对应点变红）。

已部署到Git Pages，可以通过以下链接直接访问：
